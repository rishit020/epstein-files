Constructor logic (per D-02):
1. `self._cap = cv2.VideoCapture(device_index)`
2. Set resolution: `self._cap.set(cv2.CAP_PROP_FRAME_WIDTH, config.CAPTURE_WIDTH)` (1280)
3. Set resolution: `self._cap.set(cv2.CAP_PROP_FRAME_HEIGHT, config.CAPTURE_HEIGHT)` (720)
4. Set fps: `self._cap.set(cv2.CAP_PROP_FPS, config.CAPTURE_FPS)` (30)
5. Poll `self._cap.isOpened()` in a monotonic loop: `deadline = time.monotonic() + 5.0`, while `time.monotonic() < deadline`, sleep 0.1s between checks. If opened, break. If deadline exceeded, `self._cap.release()` then raise `SourceUnavailableError(f"Camera {device_index} not available after 5.0s")`
6. `self._frame_id: int = 0`
7. Log: `_log.info("WebcamSource opened: device=%d, resolution=%dx%d@%dfps", device_index, config.CAPTURE_WIDTH, config.CAPTURE_HEIGHT, config.CAPTURE_FPS)`

`read()` method:
```python
def read(self) -> RawFrame | None:
```
1. `try:` block wrapping everything
2. `ret, frame = self._cap.read()`
3. If `not ret or frame is None`: `_log.debug("cap.read() failed frame_id=%d", self._frame_id)` then `return None`
4. `ts = time.monotonic_ns()`
5. `h, w = frame.shape[:2]`
6. Build and return: `RawFrame(timestamp_ns=ts, frame_id=self._frame_id, width=w, height=h, channels=frame.shape[2], data=frame, source_type='webcam')`
7. After building frame: `self._frame_id += 1`  (increment AFTER using current value)
8. `except Exception as exc:` `_log.error("WebcamSource.read() error: %s", exc)` then `return None`
